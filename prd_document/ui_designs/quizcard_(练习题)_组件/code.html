What is the integral of 2x dx?

Explanation

To find the integral of 2x, we use the power rule for integration, which states that the integral of xⁿ is (xⁿ⁺¹)/(n+1). In this case, n=1.

Applying the rule: ∫2x dx = 2 * ∫x¹ dx = 2 * (x¹⁺¹ / (1+1)) + C = 2 * (x²/2) + C = x² + C.

The 'C' represents the constant of integration, which is necessary for indefinite integrals.
