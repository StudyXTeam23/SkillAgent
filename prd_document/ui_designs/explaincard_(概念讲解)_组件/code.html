The Chain Rule

In calculus, the chain rule is a formula to compute the derivative of a composite function. If you have a function f(x) and another function g(x), the composite function can be written as f(g(x)).

The chain rule states that the derivative of this composite function is:

d/dx [f(g(x))] = f'(g(x)) * g'(x)

This can be understood as "the derivative of the outer function (with the inner function left unchanged) times the derivative of the inner function."

Examples

1Find the derivative of y = (x² + 1)³

  • Let u = x² + 1 (inner function), so y = u³ (outer function).
  • Find the derivative of the outer function: dy/du = 3u²
  • Find the derivative of the inner function: du/dx = 2x
  • Apply the chain rule: dy/dx = (dy/du) * (du/dx)
  • Substitute back: dy/dx = (3u²) * (2x) = 3(x² + 1)² * 2x
  • Simplify: dy/dx = 6x(x² + 1)²

2Find the derivative of y = sin(3x)

  • Let u = 3x (inner function), so y = sin(u) (outer function).
  • Derivative of the outer function: dy/du = cos(u)
  • Derivative of the inner function: du/dx = 3
  • Apply the chain rule: dy/dx = cos(u) * 3
  • Substitute back: dy/dx = 3cos(3x)
